# Scarlet
> This project is experimental and intended for learning.
Scarlet is a programming language made to learn how to design languages and to learn how lexers, parsers, or anything work behind the scenes.
## Philosophy
- It is made step by step to comprehend how everything works before adding something new
- It's development will be incremental: I'll be adding new things to it over time.
- Readable code easy to understand
- Designed to be easy to learn
## Current status
- Lexer: in development
- Parser: planned
- Variables: planned
- Functions: planned
## Examples
As you can see, there are 2 codes: demo and main. Demo is the last version I used, when main gets slightly stable, I'll delete it
Command (demo):
```Scarlet - demo
say hello world
```
OUTPUT:
```OUTPUT
hello world
```
Command (main):
```Scarlet - main
say("hello world");
```
OUTPUT (Current):
```OUTPUT
command: say
```
OUTPUT (in the future):
```OUTPUT
hello world
```
